# FROM K CAR

관심있는 중고차 매물(주로 KB차차차 등)을 한 곳에 모아 보고 관리하는 **개인용 매물 아카이브 웹사이트**입니다.
관리자(비밀번호 인증)만 매물 URL을 입력해 차량 정보(차량명·가격·연식·주행거리·옵션·이미지·소개글 등)를 자동으로 추출·등록할 수 있고, 방문자에게는 한국어/러시아어 병기로 매물을 소개합니다.

---

## 1. 완료된 기능

### 🏠 메인 갤러리 (`index.html`)
- 검은색 상단 헤더 + 새 로고(흰 글자 버전, `images/from-k-car-logo-white.jpg`) 적용
- 히어로 영역 메인 타이틀(한국어) + 동일 문장의 **러시아어 번역**을 함께 표기
  - 한글: "한국에서 믿을 수 있는 중고차 매물, 프롬케이카 김종찬 팀장에게 맡겨주세요. 원하시는 매물, 가장 빠르게 찾아드립니다."
  - 러시아어: 동일 의미의 번역 문장을 이탤릭 + 좌측 보더 스타일로 하단에 표기
- 등록된 매물을 카드형 UI로 표시 (대표 이미지, 가격, 연식/주행거리/연료/지역 스펙 칩, 상태 배지)
- 상단 히어로 영역에 요약 통계(총 매물 수 / 평균 가격 / 평균 주행거리 / 브랜드 종류) 표시
- 검색(차량명·지역·번호판), 브랜드/연료 필터, 정렬(최근등록·가격·주행거리·연식) 기능
- 카드를 클릭하면 상세 모달이 열리며, 여러 장의 차량 이미지를 좌우 화살표/썸네일/키보드(←→)로 넘겨볼 수 있는 이미지 갤러리 제공 (모달은 화면에 맞게 축소된 크기로 표시되며, 닫기(X) 버튼과 팝업 바깥 영역 클릭 시 자동 닫힘 지원)
- 상세 모달에서 기본정보(연식/주행거리/연료/변속기/배기량/색상 등), 사고·보험 이력, 진단 요약, 주요 옵션, **매물 소개(한글 + 러시아어 번역 병기)**, 원본 매물 링크 확인 가능
- **딜러(담당자) 정보 노출**: 히어로 영역에 담당자 사진·이름(김종찬 팀장)·전화번호(010-7449-8282, 강조 표시)를 담은 프로필 카드를 표시하며, 전화번호/전화 상담 버튼 클릭 시 바로 전화 연결(`tel:`)됩니다.
- **ADMIN 메뉴로 진입점 통일**: 헤더 우측 상단의 "ADMIN" 메뉴를 클릭하면 "관리자 로그인", "매물 추가" 항목이 드롭다운으로 나타나며, 모두 `admin.html`로 이동합니다. 메인 화면 중앙에 별도로 노출되던 "관리자 로그인"/"매물 추가" 버튼은 제거되어 진입 경로가 이 메뉴 하나로 통일되었습니다.

### 🔐 관리자 페이지 (`admin.html`)
- **비밀번호 로그인 게이트**: 페이지 진입 시 전체 화면 잠금 화면이 먼저 표시되며, 올바른 비밀번호를 입력해야 관리자 콘텐츠(URL 자동수집, 매물 목록 관리 등)에 접근할 수 있습니다.
  - 관리자 비밀번호: `4887asdf`
  - 로그인 상태는 브라우저의 `sessionStorage`에 저장되어, 같은 탭에서는 새로고침해도 유지되고 "로그아웃" 버튼 또는 탭 종료 시 해제됩니다.
  - ⚠️ **중요 보안 안내**: 이 비밀번호 게이트는 **클라이언트(JavaScript) 측 검증**이며, 서버 단에서 요청을 차단하는 진짜 인증이 아닙니다. 개발자 도구 등으로 우회될 여지가 있으므로, 더 강력한 보호가 필요하다면 **Publish 단계의 Access Rules(호스팅 접근 제어)** 로 `admin.html` 경로 자체를 인증된 사용자만 접근 가능하도록 별도 설정하는 것을 권장합니다.
- **URL 자동수집**: 로그인 없이 공개적으로 열람 가능한 매물 상세페이지 URL을 입력하면, 브라우저에서 직접 페이지 내용을 가져와 다음 정보를 자동으로 추출합니다.
  - 차량명, 브랜드, 차량번호(번호판), 판매가격, 연식, 주행거리, 연료, 변속기, 배기량, 색상/시트색상, 지역
  - 사고·보험 이력 요약, 진단(프레임/외부패널)·주행거리 분석 요약
  - 주요 옵션 목록
  - 차량 이미지 전체 목록 (대표 이미지 자동 선정, 나머지는 썸네일로 선택 가능)
  - 이 기능은 **관리자 로그인 이후에만** 접근할 수 있는 화면(`admin.html`) 안에만 존재하며, 갤러리(`index.html`)에는 노출되지 않습니다.
- **매물 소개(한글/러시아어) 직접 입력**: 자동 추출로는 채워지지 않는 "매물 소개" 문구를 한글과 러시아어 번역으로 각각 입력할 수 있는 전용 입력창을 제공합니다. 입력한 내용은 갤러리 상세 모달에 한글 원문 + 러시아어 번역 순으로 표시됩니다.
- 추출된 내용은 **등록 전에 모든 항목을 직접 수정**할 수 있는 폼으로 제공됩니다 (자동 추출이 실패하거나 부정확한 경우 직접 입력/보정 가능)
- "직접 입력으로 추가" 버튼으로 URL 없이도 매물을 수동 등록할 수 있음
- 등록된 매물 목록 테이블에서 **수정 / 삭제** 관리 가능

### ⚙️ 자동수집 동작 원리 (중요)
- 브라우저 JavaScript는 보안(CORS) 정책상 다른 사이트에 직접 접속할 수 없기 때문에, **읽기 전용 리더 서비스(r.jina.ai)를 경유**하여 대상 페이지의 텍스트 콘텐츠를 가져온 뒤, 정규식 기반 파서(`js/parser.js`)로 필요한 필드를 추출합니다.
- **로그인 후에만 열람 가능한 페이지, 혹은 자동화 접근을 차단한 페이지는 자동 추출이 되지 않습니다.** 이 경우 관리자 페이지에서 "직접 입력"으로 수동 등록해야 합니다.
- 현재 KB차차차(kbchachacha.com)의 **공개 매물 상세페이지 구조**를 기준으로 파서가 작성되어 있습니다. 다른 사이트나 페이지 구조가 바뀌면 추출이 정확하지 않을 수 있어 수동 보정이 필요할 수 있습니다.

## 3. 페이지 경로 (Entry URI)

| 경로 | 설명 |
|---|---|
| `index.html` | 메인 매물 갤러리 (누구나 열람 가능) |
| `admin.html` | 관리자 페이지 — **비밀번호 로그인(`4887asdf`) 후에만** 매물 URL 자동수집 등록, 목록 수정/삭제 가능 |

## 5. 기술 구성

- **프론트엔드**: 순수 HTML/CSS/JavaScript + Tailwind CSS(CDN), Font Awesome 아이콘
- **자동추출 파서**: `js/parser.js` — r.jina.ai 리더 서비스를 통해 페이지를 가져오고 정규식으로 필드 파싱
- **API 래퍼**: `js/api.js` (Table API 호출), `js/common.js` (토스트/포맷 유틸)
- **페이지 로직**: `js/main.js` (갤러리), `js/admin.js` (관리자)
- **관리자 인증**: `js/auth.js` — 클라이언트 측 비밀번호 게이트 (`sessionStorage` 기반)

```

admin.html           관리자 페이지
css/style.css        공통 스타일
js/parser.js          URL 자동수집 파서
js/api.js             Table API 래퍼
js/common.js          공통 유틸(토스트, 포맷터)
js/main.js            메인 페이지 로직
js/admin.js           관리자 페이지 로직
```

## 6. 아직 구현되지 않은 기능 / 알아두어야 할 제약

- **로그인 전용 매물 수집 불가**: 로그인해야만 보이는 매물 페이지는 브라우저 보안 정책상 자동 수집할 수 없습니다. (서버사이드 인증/크롤링은 정적 웹사이트로 구현 불가)
- **다른 사이트 지원 미흡**: 현재 파서는 KB차차차 페이지 구조 기준으로 작성되어, 엔카·보배드림 등 다른 사이트는 필드 추출이 정확하지 않을 수 있습니다.
- **가격 변동 추적/알림 기능 없음**: 등록 이후 원본 매물의 가격 변경 등을 자동으로 재확인하지 않습니다. (재수집하려면 관리자 페이지에서 다시 URL 입력)
- **관리자 인증은 클라이언트 측 검증**: `admin.html`의 비밀번호 로그인(`4887asdf`)은 JavaScript로 구현되어 있어, 진짜 서버 단 인증만큼 강력하지 않습니다. 완전한 보호가 필요하면 Publish의 Access Rules 기능으로 `admin.html` 경로 자체를 제한하는 것을 권장합니다.
- **매물 소개 자동 번역 없음**: `description_ko`/`description_ru`는 관리자가 직접 입력하는 값이며, 사이트가 자동으로 번역해주지는 않습니다.
- **이미지 자체 저장 없음**: 이미지는 원본 사이트(kbchachacha.com)의 URL을 그대로 참조합니다. 원본 이미지가 삭제되면 표시되지 않을 수 있습니다.

## 7. 추천 다음 단계

1. `admin.html`에 Publish Access Rules(서버 단 접근 제어)를 추가 적용해 클라이언트 측 비밀번호 게이트를 보완
2. 여러 매물 사이트 지원을 위한 파서 확장 (엔카 등)
3. 매물별 "관심도/우선순위" 태그, 즐겨찾기 기능 추가
4. 가격 변동 히스토리 기록 (재수집 시 이전 가격과 비교)
5. 다중 URL 일괄 등록 기능 (여러 줄 붙여넣기)
6. 매물 소개 러시아어 번역을 자동화(번역 API 연동 등)하는 방안 검토
